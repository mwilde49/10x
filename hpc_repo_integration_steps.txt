HPC Repo Integration Steps for 10x Pipeline Support
====================================================

Prerequisites:
- The 10x repo is live at https://github.com/mwilde49/10x (tagged v1.0.0)
- All HPC integration files are staged in the 10x repo under hpc_updates/
- The hpc_updates/ directory mirrors the HPC repo's directory structure exactly

Overview of Changes:
- 7 modified files (replacements for existing HPC repo files)
- 3 new SLURM templates
- 3 new config templates
- 1 new git submodule

All changes have been risk-hardened:
- PIPELINE_CONTAINERS map is unchanged (only addone/bulkrnaseq/psoma)
- is_known_pipeline() preserves original PIPELINE_CONTAINERS check via OR
- get_container_path() body is identical to current production
- tjp-setup uses explicit per-pipeline blocks (no generic loop refactor)
- manifest.json schema is stable (same field names for all pipelines)
- manifest.sh has no dependency on common.sh functions
- 10x submodule is not sourced by any shared infrastructure file

================================================================================
STEP 1: Add the 10x submodule
================================================================================

git submodule add https://github.com/mwilde49/10x containers/10x
cd containers/10x && git checkout v1.0.0 && cd ../..
git add containers/10x .gitmodules

================================================================================
STEP 8: On HPC — install 10x tools and stage references
================================================================================

On Juno HPC:

a) Extract 10x tool tarballs to /groups/tprice/software/:
   - /groups/tprice/software/cellranger/
   - /groups/tprice/software/spaceranger/
   - /groups/tprice/software/xeniumranger/

b) Build Cell Ranger reference (one-time):
   cellranger mkref \
     --genome=GRCh38 \
     --fasta=/path/to/GRCh38.primary_assembly.genome.fa \
     --genes=/path/to/gencode.v48.primary_assembly.annotation.gtf
   Move output to /groups/tprice/pipelines/references/cellranger/refdata-gex-GRCh38-2024-A

c) Pull updated HPC repo:
   cd /groups/tprice/pipelines
   git pull origin master
   git submodule update --init --recursive

d) Run smoke tests:
   bash containers/10x/test/test_cellranger.sh /groups/tprice/software/cellranger
   bash containers/10x/test/test_spaceranger.sh /groups/tprice/software/spaceranger
   bash containers/10x/test/test_xeniumranger.sh /groups/tprice/software/xeniumranger

e) Run setup and launch test:
   tjp-setup
   tjp-test cellranger
   tjp-test-validate cellranger
